SwitchDesktops

Virtual desktop switcher with crossfade transitions for Windows 10/11.

Install for Windows

Requirements

  • Windows 10 build 19041+ or Windows 11 (x64)
  • You may need to trust the signing certificate on first install — see below.

First-time install

If this is a self-signed build, download SwitchDesktops.cer, right-click → Install Certificate → Local Machine → place in Trusted People. Then click Install above.

Updates

Once installed, the app checks this URL on every launch and auto-installs updates in the background.
